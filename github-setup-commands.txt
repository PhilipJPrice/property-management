Git Setup/Initialize Repository
--------------------------
1. git init
2. git add .
3. git commit -m "message"
4. git remote add origin https://github.com/USERNAME/REPO.git
5. git remote set-url origin https://github.com/USERNAME/REPO.git
6. git push -u origin main

Add to Repository
--------------------------
1. git pull
2. git add .
3. git commit -m "message"
4. git push
